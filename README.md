# OKR Tree Builder

Multi-department OKR tree with Company Overview, department trees, and OKRA chatbot. Built for the Continuous Discovery Habits framework (Teresa Torres).

## Open the app (shareable link)

**Use this URL to open the app in your browser:**

**https://htmlpreview.github.io/?https://raw.githubusercontent.com/eyalsta-stack/OKR_Tree/main/okr-builder.html**

> Use the **raw** GitHub URL in htmlpreview (raw.githubusercontent.com), not the blob page (github.com/.../blob/...). The raw URL returns the actual HTML file so the app can run.

## What you’ll see

1. **Company Overview** – Opens first. You’ll see the company tree with the pre-stored **FEF → Communications** OKR tree.
2. **All Trees (gallery)** – Click “All Trees” to open the gallery. Click **FEF - Communications** → **Open** to open that department’s tree on the canvas.
3. **Department tree**
   - **Click the objective name** (or the 🔗 badge) → opens the “Shared with sub-departments” panel.
   - **Click + / −** → expand or collapse Key Results.
   - **Click the pencil** → edit the node.
4. **+ New Tree** – Upload a strategy doc (or paste/URL), add an OpenAI API key, and generate a new tree.

## Files

- **okr-builder.html** – Main app (company view, department trees, gallery, OKRA, shared-depts panel).
- **okr-tree.html** – Standalone Communications OKR tree (single file, no backend).

## Run locally

Open `okr-builder.html` in your browser (double-click or drag into the browser), or serve the folder and open `http://localhost:PORT/okr-builder.html`.
